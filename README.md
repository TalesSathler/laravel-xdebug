Simple project with xdebug.


### Project

- Laravel 10
- Docker
- Php8.3-fpm
- Postgres
- Nginx
- Redis
- Xdebug -> docker/app/xdebug.ini
- VScode configuration -> docker/xdebug/vscode.json


### How to run
- cd docker
- sh build.sh
